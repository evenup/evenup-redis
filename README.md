What is it?
===========

A Puppet module that manages Riak.

Released under the Apache 2.0 licence

Usage:
------

To install:
<pre>
  class { 'riak': }
</pre>

Lots of config parameters: [init.pp](https://github.com/evenup/evenup-riak/blob/master/manifests/init.pp)
